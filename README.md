# Bank-Data-Tables
Repository to create and refresh tables in `EDS.SB_DATA_SCIENCE` tables.

# Usage
This library needs two files for creating/refreshing the table:
1. SQL query in a `.sql` file to pull the data from snowflake.
2. A `.csv` file with the list of columns for the table.

Add the sql to pull data from `EDS.SB_CREDIT_ANALYTICS.MODEL_EXECUTION_LOG` table to the `sql` folder.\
Example sql:
```
select
    MODEL_EXECUTION_CONTEXT:acaps_ref_id acap_key,
    MODEL_EXECUTION_TIMESTAMP:: timestamp as MODEL_EXECUTION_TIMESTAMP,
    model_infrastructure_id as EXL_Container_Version,
    f.key as feature,
    f.value as value
from sb_credit_analytics.model_execution_log mel,
    LATERAL FLATTEN(INPUT => mel.model_input:Feature) AS f
where
    MODEL_ID = 'BAT-2' 
    and MODEL_EXECUTION_CONTEXT:shadow = 'false'
    and MODEL_EXECUTION_TIMESTAMP > %s;
```
The query needs to have a timestamp in the where field for dynamic limiting of the data to be pulled.
Columns `ACAP_KEY`,  `MODEL_EXECUTION_TIMESTAMP` and `EXL_Container_Version` are mandatory in the query.\
The `.csv` file needs to be placed in `table_columns` folder.\
